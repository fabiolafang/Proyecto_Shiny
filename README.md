# Proyecto Shiny


### Contexto 
Este proyecto se enmarca en el desarrollo de una aplicación analítica interactiva utilizando Shiny para Python, integrando metodologías de ciencia de datos con principios de desarrollo colaborativo mediante Git.


### Objetivo 
Diseñar e implementar una plataforma interactiva que permita el análisis exploratorio de datos cinematográficos, facilitando la identificación de patrones y tendencias en la industria del cine.

- Existe una correlación positiva entre presupuesto y revenue
